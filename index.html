Dashboard Prediksi Remaining Useful Life (RUL) Gas Engine

Oleh: Daffa Alauddin | Repositori Tugas Akhir

Dashboard ini dikembangkan sebagai pelengkap penelitian tugas akhir untuk mendemonstrasikan sistem pipeline prediksi Remaining Useful Life (RUL) pada Kompresor Gas Engine berbasis Deep Learning menggunakan fitur temperatur data lapangan.

📁 File Explorer & Akses Dataset

Bagi mahasiswa penerus penelitian atau pihak prodi yang ingin meninjau, mereplikasi, atau melanjutkan pengembangan kode program dan dataset mentah, silakan gunakan tautan resmi di bawah ini:

📂 Buka File Explorer di GitHub 📥 Download Semua File (.ZIP)

Struktur Berkas Utama Proyek:

  • 📁 Data RTF Mentah/ – Folder berisi berkas data mentah log lapangan (.rtf).
  • 📁 Data Siap Tampil/ – Folder berisi visualisasi dataset hasil ekstraksi dan filter.
  • 📁 K101 Data/ – Folder data logger parameter temperatur utama kompresor gas engine (.csv).
  • 📁 Preprocess/ – Struktur skrip pembantu untuk pembersihan dan rekayasa fitur data.
  • 📁 Testing/ – Lingkungan pengujian model deep learning beserta data validasi.
  • 📁 Training/ – Folder berisi kode latih arsitektur deep learning dan penyimpanan bobot model.
  • 📄 Testing/GUI_New.py – Kode antarmuka grafis (GUI) siap pakai terintegrasi dengan arsitektur model final.
  • 📄 requirements.txt – Daftar pustaka dependency environment Python yang wajib diinstal.

📊 Visualisasi & Hasil Metrik Evaluasi Model

Ringkasan performa akurasi model dalam memprediksi sisa waktu pakai operasional mesin sebelum mengalami kegagalan (failure):

Metrik Pengujian Nilai Hasil Uji (Testing)
