Gas Engine Remaining Useful Life (RUL) Prediction Dashboard

By: Daffa Alauddin | Undergraduate Thesis Repository

This dashboard was developed as a complement to an undergraduate thesis research to demonstrate a Deep Learning-based Remaining Useful Life (RUL) prediction pipeline for a Gas Engine Compressor using real field temperature logger features.

Main Project Repository Structure:

  • 📁 Data RTF Mentah/ – Folder containing raw field log files (.rtf).
  • 📁 Data Siap Tampil/ – Folder containing visualizations of extracted and filtered datasets.
  • 📁 K101 Data/ – Folder containing the main temperature parameter logger datasets for the gas engine compressor (.csv).
  • 📁 Preprocess/ – Directory structure for data cleaning and feature engineering scripts.
  • 📁 Testing/ – Evaluation environment for the deep learning models along with validation data.
  • 📁 Training/ – Folder containing deep learning model training configurations and saved weights.
  • 📄 Testing/GUI_New.py – Ready-to-use Graphical User Interface (GUI) script integrated with the final model architecture.
  • 📄 requirements.txt – List of mandatory Python environment library dependencies.
